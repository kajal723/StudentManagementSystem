# StudentManagementSystem
Java-based Student Management System using Swing GUI, OOP concepts (Inheritance, Polymorphism, Interfaces, Exception Handling), Collections, Generics, and Multithreading. Includes DAO classes, MySQL database integration, and JDBC connectivity for performing CRUD operations efficiently.
